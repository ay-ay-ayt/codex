# Study Timer

Small Electron study timer with an always-on-top compact window.

## Run

1. `npm.cmd install`
2. `electron-start.bat`

You can also start it from the terminal with `npm.cmd run start`.

## Files

- `src/`: timer UI and local state
- `electron/`: Electron main process and preload bridge
- `electron-start.bat`: build and launch helper for Windows
- `THIRD_PARTY_NOTICES.md`: runtime dependency notices for safer redistribution
