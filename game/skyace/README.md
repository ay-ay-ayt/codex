# skyace

Browser-playable Sky Ace Duel 3D files.

Included in this folder:
- `index.html`, `main.js`, `style.css`, and `manifest.webmanifest`
- local `vendor/three.module.min.js`
- local cockpit art and required Poly Haven texture subset under `assets/`
- Poly Haven license notices under `assets/_licenses/`
