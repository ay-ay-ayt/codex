# Game Hub

This directory holds multiple browser-game projects inside the `ay-ay-ayt/codex` repository.

## Games

- `robot-battle-kai/` - 3D robot air-combat boss battle vertical slice
- `skydive/` - Rebuilding from scratch. The next version will be a new browser skydive project.

## Notes

- Each game should keep its own `assets/`, `src/`, `vendor/`, `README.md`, `THIRD_PARTY.md`, and `CREDITS.md` inside its own folder.
- GitHub Pages entry points should live inside each game folder.
- Each game should also include a simple local launch script such as `起動.bat` so it can be started easily.
